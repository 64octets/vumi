Super-scalable

Vumi is built for scale. It is designed from the ground up for cloud and elastic computing environments allowing you to scale your application horizontally on demand.

Multiple territories

Want to run mobile conversations across multiple territories? Vumi is ready. Connect to multiple networks in various countries all at the same time.

Multiple applications

Different audience, different applications, one platform. Is your target audience on MXit or do they prefer SMS or even USSD? Vumi is ready, it will happily engage in conversations across all.

Open source

The source code for Vumi is available under the very liberal BSD license. You can download and run Vumi on your own servers at no cost.

Easy integration

Integration with application is a breeze. Vumi's simple web based APIs allow you to extend your existing applications and engage in conversations with your audience.

Economies of scale

Vumi scales financially. You can host conversations with a large or small audience and you only pay for the actual messages you send and receive.
